[Read_Manual]

DECRYPTOR generates a secret 4-digit code. Crack it through logical deduction before giving up.

// Constraints

All 4 digits must be unique. The first digit cannot be 0.

// How to Enter a Guess

Tap the on-screen Numpad to enter digits. On desktop, you can also type on your keyboard. Press > Inject_Code to submit.

// Feedback After Each Guess

Each guess in the Attempt Log shows two clues:

Correct Number — how many of your digits exist anywhere in the secret code.
Position — how many are also in the exact right slot.
// Example

Secret = 5 2 3 4  |  Guess = 4 2 7 8

4 and 2 are both in the code → Correct Number: 2
Only 2 is in the right slot (slot 2) → Position: 1
// Hint

One free Scan_Hint per game — reveals the exact value of one digit at a random position.

// Numbers for Reference

Tap any digit in the Numbers for Reference grid to cross it out. Tap again to restore.

// Terminate

Press > Terminate to give up and reveal the code. A new game starts in 3 seconds.
